About this method
This is a common field-style approximation used when you only have four side measurements. It works best when the plot is close to a rectangle (or generally “regular”).
  • Pros: very easy, needs only 4 sides.
  • Cons: not exact for irregular plots because angles/shape are unknown.
  • Use case: quick estimate for land deals and rough measurement.
Equation

About this formula
Brahmagupta’s area formula is historically attributed to the Indian mathematician Brahmagupta (7th century). It gives the area of a cyclic quadrilateral (a 4-sided shape whose corners lie on a circle).
  • Exact only when the quadrilateral is cyclic.
  • For general land plots it may not match reality, but it is useful to compare.
  • If this result shows “Not possible”, the cyclic assumption fails for your sides.
Equation
s = (a + b + c + d) / 2 Area_sqft = √[(s-a)(s-b)(s-c)(s-d)] Shotok = Area_sqft / 435.6

What “maximum possible” means
If you fix the four side lengths, many different quadrilateral shapes are possible. Among those, the largest area happens when the quadrilateral is cyclic. That’s why this value matches the cyclic (Brahmagupta) area.
  • Think of this as the upper limit of area for the same sides.
  • Your real land area should usually be ≤ this maximum.
Equation
MaxArea_sqft = √[(s-a)(s-b)(s-c)(s-d)] (where s = (a + b + c + d) / 2) MaxShotok = MaxArea_sqft / 435.6
Shotok is calculated as: Shotok = Square Feet ÷ 435.6.
